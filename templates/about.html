How WCA StatBot works

What is WCA StatBot?

WCA StatBot is an AI-powered tool that lets you query World Cube Association competition data using plain English. Instead of writing SQL, just ask a question and get instant results.

How It Works

1

Ask a Question

Type any question about WCA data in natural language.

2

AI Translation

Anthropic's Claude AI translates your question into a SQL query against the WCA database.

3

Get Results

The query runs against the WCA database and results are displayed in a formatted table.

Database

The app queries a database populated from the official WCA data export (March 14, 2026), containing:

281,645 Competitors
6,346,883 Results
17,150 Competitions
21 Events

Example Questions

  • What is the world record for 3x3?
  • Who are the top 10 fastest 2x2 solvers?
  • How many competitions have been held in the United States in 2025?
  • Who has the most world record single results?
  • Who is the top 3x3 competitor with the first name "Timothy"?
  • Who placed first in 3x3 finals at the 2023 World Championship?

Tips for Better Results

  • Be Specific

    Make sure to be as precise as possible when formulating your question so the AI knows exactly what you're looking for, i.e., specify between single and average results.

  • Use Clear Language

    Avoid ambiguous terms and use clear, unambiguous language when asking your question, i.e., use "United States" instead of "America".

  • Understand how WCA Results are Structured

    Having a basic understanding of WCA results and competition data can help you ask more effective questions and interpret the results.

  • Experiment with Different Phrasings

    Try rephrasing your question in different ways to see if you get more accurate or detailed results.. i.e, ask "Who placed first in 3x3 finals" instead of asking "who won the competition".

Discord Bot

WCA StatBot is also available as a Discord bot, bringing the same natural language query capabilities directly into your server!

Commands

!wca query <question> Ask a question about WCA data. Aliases: !wca q, !wca ask
!wca help Show available commands and example queries
!wca ping Check bot latency

Add to Your Server

1

Click the Invite Link

Use this link to add WCA StatBot to your Discord server.

2

Select Your Server

Choose the server you want to add the bot to from the dropdown. You must have Manage Server permissions in that server.

3

Authorize Permissions

Review and approve the requested permissions, then click Authorize. The bot only needs permission to send messages.

4

Start Querying

Head to any text channel and type !wca query followed by your question. For example: !wca q Who has the fastest 3x3 single?
